# Replace


## Supported Types

### `boolean`

```typescript
const value: boolean = false;
```

### `string`

```typescript
const value: string = "<value>";
```

### `number`

```typescript
const value: number = 9446.69;
```

### `any[]`

```typescript
const value: any[] = [
  "<value>",
];
```

### `{ [k: string]: any }`

```typescript
const value: { [k: string]: any } = {
  "key": "<value>",
};
```
